Scrabble
  • Age: 5
  • Minimum players: 2
  • Category: Word game
  • Type: Board-game
  • Restrictions: Large vocabulary
  • Target audience: All ages
Scrabble is a word game in which two to four players score points by placing tiles, each bearing a single letter, onto a game board divided into a 15×15 grid of squares. The tiles must form words that, in crossword fashion, read left to right in rows or downward in columns, and be included in a standard dictionary or lexicon. The game is played by two to four players on a square game board imprinted with a 15×15 grid of cells (individually known as "squares"), each of which accommodates a single letter tile. In official club and tournament games, play is between two players or, occasionally, between two teams, each of which collaborates on a single rack.
